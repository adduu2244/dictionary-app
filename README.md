# 📖 Dictionary App

A simple and responsive dictionary web application that allows users to search for any English word and get its meaning, example, synonyms, and pronunciation.

---

## 🚀 Features

- 🔍 Search any English word
- 📚 Get word meaning instantly
- 🧠 View synonyms (up to 3)
- 🗣️ Listen to pronunciation (audio)
- ✍️ Example sentence display
- 🎨 Clean and modern UI design


## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Free Dictionary API (dictionaryapi.dev)


## ⚙️ How It Works

1. Enter a word in the search box
2. Click the **Search** button
3. The app fetches data from API
4. Displays:
   - Meaning
   - Example
   - Part of Speech
   - Synonyms
   - Audio pronunciation
